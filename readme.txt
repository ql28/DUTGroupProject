Voici la dernière version du logiciel adec_56, l'éditeur de plan de feu (et de salle) de théâtre. 

Pour lancer l'application, vous devez posséder la version 1.8 de java :

http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html

Si vous êtes sous Windows, vous devez sur le fichier Adec56_c.jar.
Si vous êtes sous Linux, lancez un terminal, placez-vous dans le répertoire de l'application et exécutez
la commande : java -jar Adec56_c.jar

Pour plus d'informations, consulter l'aide dans le logiciel ou n'hésitez pas à contacter M. Merciol 
ou notre équipe.
