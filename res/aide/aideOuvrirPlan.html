Ouvrir un plan

Pour ouvrir un plan, cliquez sur le bouton , puis sélectionnez le plan à ouvrir dans votre répertoire personnel (.pf ou .ps).

Vous pouvez aussi double cliquer sur un plan dans le menu de gauche, le glisser-déposer dans l'interface ou faire clique droit, Ouvrir. Utilisez la recherche par nom pour filtrer les plans déjà disponible dans dans le logiciel, ainsi que ceux que vous avez créés. Il est possible d'accéder à ces fichiers dans les répertoires "local" et "serveur" du logiciel.
